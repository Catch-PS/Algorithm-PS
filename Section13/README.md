# 문제 풀이 방법

- 풀이 시간 체크(1시간 초과시 초과로만 기재)
- 컴퓨팅사고 2,3줄로 간략히 요약
- 자신이 생각하는 시간복잡도
- 일요일 밤에 참석여부 투표 
- 스터디전에 전주차 문제들 복습
